DashJazz IoT Connector

The Dashboard for the Internet Of Things

PubNub introduces the simplest dashboard ever. Connect all your devices and talk to them in real time. By using PubNub, you can build these dashboards with just a few lines of code. Get out your favorite embedded device and start connecting! This dashboard lets you connect an Arduino to a Raspberry Pi, lets them talk bidirectionally to each other and you can view which device is online at any instant.

Github Zip Tutorial

This demo shows you PubNub Data Sync in action. Punch in your sub and pub keys and start the demo. Click on the light, music, door and garage icons to see how you can change the state of variables and update them from anywhere.
